Detailed Report on the RomCom Ransomware Actor

Executive Summary

The RomCom threat actor, also tracked as Storm-0978, UAT-5647, Tropical Scorpius, and Void Rabisu, is a Russia-aligned cyber espionage group that has evolved into a hybrid actor blending state-sponsored intelligence gathering with financially motivated ransomware operations. Active since at least 2022, RomCom initially targeted Ukrainian and NATO-affiliated entities but expanded to private sectors in Europe, North America, and the UK by 2025. The group employs sophisticated phishing, zero-day exploits, and custom malware like SnipBot for initial access, persistence, and data exfiltration. While ransomware deployment (e.g., Cuba and Underground strains) supports double-extortion tactics, recent activities emphasize espionage over immediate financial gain. No credible references linking RomCom to Netwalker were identified in available sources; any perceived connection may stem from misattribution or unrelated ransomware nomenclature.

Overview and Aliases

RomCom is described as an evolving, sophisticated threat actor group specializing in espionage and financially motivated attacks. The group aligns with Russian strategic interests, targeting military personnel, government agencies, political leaders, and private industries in Ukraine, Poland, NATO countries, the UK, Europe, Canada, and North America. Key aliases include:

  • Storm-0978 (Microsoft attribution)
  • UAT-5647
  • Tropical Scorpius
  • Void Rabisu (Trend Micro)

First observed in 2022, RomCom's operations demonstrate adaptability, shifting from regional espionage to global hybrid threats by 2025.

History and Evolution

RomCom emerged in 2022 with campaigns spoofing popular apps to target Ukrainian military entities, abusing tools like KeePass and SolarWinds for credential theft and persistence. By mid-2023, the group conducted phishing against defense and government sectors in Europe and North America, using lures tied to NATO and Ukrainian politics. This marked a pivot from pure espionage to include financial motives, with ransomware deployments noted in late 2023.

In 2024, RomCom exploited high-profile zero-days, such as CVE-2023-36884 (Microsoft Word RCE) and CVE-2024-9680 (Firefox use-after-free, chained with CVE-2024-49039 for privilege escalation). Targets broadened to English-speaking regions, with malware variants like PEAPOD (RomCom 4.0) enhancing evasion. By 2025, the actor aggressively targeted private sectors—retail, hospitality, critical infrastructure, financial, manufacturing, defense, and logistics—via spear-phishing and compromised portals. This evolution reflects a convergence of cyberwarfare and cybercrime, with RomCom maintaining ties to a Russia-linked ecosystem.

Tactics, Techniques, and Procedures (TTPs)

RomCom's TTPs map to the MITRE ATT&CK framework, emphasizing social engineering and exploitation:

  • Initial Access (TA0001): Spear-phishing attachments (T1566.001) with malicious .exe disguised as .pdf, weaponized Office docs, or password-protected .rar/.zip archives; drive-by compromises (T1189) via URLs like gdrive-share[.]online; abuse of external portals (e.g., customer feedback forms).
  • Execution (TA0002): User execution of malicious files (T1204.002); exploitation of public-facing apps (T1190), including zero-days like CVE-2025-8088 (WinRAR path traversal).
  • Persistence (TA0003): DLL search order hijacking (T1574.001); registry run keys/startup folders (T1547.001); COM hijacking (e.g., CLSID registry manipulation).
  • Privilege Escalation (TA0004): Exploitation for client execution (T1203), e.g., chained Firefox/Windows vulns.
  • Defense Evasion (TA0005): Code signing abuse (T1553.002) with fraudulent certificates; virtualization/sandbox evasion (T1497.001) via registry checks (e.g., ≥55/100 recent documents in HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\RecentDocs; Shell Bags); window message-based obfuscation.
  • Credential Access (TA0006): SAM dumping (T1003.002).
  • Discovery (TA0007): System information gathering (T1082); network reconnaissance (e.g., domain controller info via AD Explorer).
  • Lateral Movement (TA0008): SMB/Windows admin shares (T1021.002, SMBExec); WMI (T1047, WMIExec).
  • Collection (TA0009): Data from local system (T1005).
  • Exfiltration (TA0010): Over C2 channels (T1041), targeting files like .pdf, .doc, .db, .mp4.
  • Impact (TA0040): Data encrypted for impact (T1486) via ransomware.

C2 infrastructure includes domains like xeontime.com, cethernet.com, and IPs such as 91.92.250.104.

Malware Variants

RomCom's modular malware ecosystem has advanced rapidly:

  • ROMCOM RAT/Backdoor: Core tool for info theft and secondary payload deployment.
  • MeltingClaw/RustyClaw: Rust-based phishing downloaders.
  • DustyHammock: Rust backdoor for persistence and C2.
  • ShadyHammock: C++ component loading XOR-encoded SingleCamper RAT DLL from registry.
  • SlipScreen: 2025 Rust/C++ loader masquerading as PDF reader; decrypts shellcode, uses COM hijacking.
  • SnipBot (RomCom 5.0/SingleCamper RAT): Multistage backdoor with 27 commands for execution, file ops, reconnaissance, and exfiltration; anti-sandbox checks, encrypted strings, dynamic API resolution; downloads tools like Plink.exe (SSH tunneling), 7-Zip (archiving), and custom DLLs (e.g., SnippingTool.dll for screenshots). Active since Dec 2023, focused on espionage.
  • PEAPOD (RomCom 4.0): Offshoot with enhanced obfuscation.
  • Mythic Agent: Cross-platform backdoor used in 2025 campaigns.

Key Campaigns

  • 2022–2023: UAT-5647 (Ukraine/Poland targets); KeePass/SolarWinds abuse (Ukraine/UK).
  • Operation Deceptive Prospect (Apr 2025): UK retail/hospitality/critical infrastructure via feedback portals.
  • Void Rabisu (2023): Financially motivated ransomware shifts.
  • SnipBot Campaigns (2024–2025): IT, legal, agriculture; post-infection recon and exfil.

Ransomware Activities

RomCom integrates ransomware into espionage ops, deploying strains like Industrial Spy, Underground, and Cuba in double-extortion schemes (encryption + data leaks). Early uses (2023) focused on financial gain, but by 2025, ransomware supports broader monetization within intelligence-driven attacks. No SnipBot incidents involved direct ransomware, indicating a tactical shift toward data theft.

Connections to Other Actors

RomCom overlaps with UNK_GreenSec (shared TransferLoader and REM Proxy on compromised MikroTik routers). Infrastructure and delivery ties exist to the Cuba ransomware ecosystem, suggesting a Russia-linked network blending state and crime actors. No references to Netwalker ransomware (active 2019–2021, focused on healthcare) were found; searches yielded no overlaps.

Recent Activities (2025)

  • WinRAR Zero-Day Exploitation (Jul–Aug 2025): CVE-2025-8088 (CVSS 8.4) via spear-phishing RAR archives disguised as job docs (e.g., CVs); targeted financial/manufacturing/defense/logistics in Europe/Canada; delivered SnipBot, RustyClaw, Mythic; patched Jul 30 after ESET disclosure. No confirmed compromises.
  • Ongoing Spear-Phishing: Job applications/complaints lures; portal abuse in UK/Europe.
  • Sector Expansion: Private industry focus, with espionage-ransomware hybrid ops.

Mitigation Recommendations

  • Patch promptly: WinRAR (≥7.13), Firefox, Windows, Microsoft Office.
  • Email filters: Block .rar/.zip attachments; scan for anomalies.
  • EDR: Monitor registry (RecentDocs, CLSID), COM hijacking, window messages.
  • User training: Verify lures (e.g., distorted PDFs).
  • Network segmentation: Limit SMB/WMI; monitor C2 domains.
  • Behavioral analytics: Detect sandbox evasion, dynamic API calls.
